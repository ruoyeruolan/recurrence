欢迎来到 GNN 与 Single Cell 分析平台

这是一个专注于图神经网络（GNN）和单细胞分析的综合平台。我们提供最先进的工具和资源，帮助研究人员进行深度学习和生物信息学分析。

通过上方的导航栏，您可以探索不同的工具和框架，包括用于图表示学习的GRACE和PyG，以及用于单细胞RNA测序分析的Seurat和Scanpy。

GRACE

图对比学习框架，用于无监督图表示学习，通过自监督对比学习获得高质量的节点嵌入。

PyG

PyTorch Geometric是一个用于图神经网络的强大库，提供了丰富的图卷积层和数据处理工具。

Seurat

R语言中用于单细胞RNA测序数据分析的综合工具包，支持数据标准化、聚类和可视化。

Scanpy

Python中用于单细胞分析的可扩展工具包，提供高效的数据处理和机器学习算法。
