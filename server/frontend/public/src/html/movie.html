📊 MovieLens 数据集理解

💡 什么是 MovieLens？

MovieLens 是一个经典的电影评分数据集，包含用户对电影的评分信息。我们将其建模为异构图。

🎯 数据集结构

👤

User 节点

610 个用户

没有特征，需要 Embedding

🎬

Movie 节点

9724 部电影

特征: 电影类型 (genres)

🔗 边的类型

边类型 源节点 目标节点 含义
('user', 'rates', 'movie') User Movie 用户给电影评分
('movie', 'rev_rates', 'user') Movie User 反向边（被谁评分）

💡 修复后的完整代码

class HeteroLinkPredictor(Module): def __init__(self, data: HeteroData, hidden: int, nlayers: int = 3): super().__init__() self.node_types = data.node_types self.edge_types = data.edge_types # Embedding层 self.embedds = ModuleDict() for ntype in self.node_types: if data[ntype].num_features == 0: self.embedds[ntype] = Embedding( data[ntype].num_nodes, 32 ) # 图卷积层 self.convs = ModuleList() for _ in range(nlayers): conv = HeteroConv({ et: SAGEConv((-1, -1), hidden) for et in self.edge_types }) self.convs.append(conv) # 输出层 self.lin = ModuleDict() for ntype in self.node_types: self.lin[ntype] = Linear(hidden, hidden) def forward(self, data: HeteroData): x_dict = {} # ✅ 正确的特征初始化 for ntype in self.node_types: if ntype in data.x_dict and data.x_dict[ntype] is not None: x_dict[ntype] = data.x_dict[ntype] elif ntype in self.embedds: idx = torch.arange(data[ntype].num_nodes) x_dict[ntype] = self.embedds[ntype](idx) else: raise ValueError( f"Node {ntype} has no features!" ) # 图卷积 for conv in self.convs: x_dict = conv(x_dict, data.edge_index_dict) x_dict = {k: v.relu() for k, v in x_dict.items()} # 输出映射 z_dict = {k: self.lin[k](v) for k, v in x_dict.items()} return z_dict
